# Nicolau Autos e Pick-ups

Landing page automotiva criada para apresentar uma empresa de veículos em União da Vitória - PR.

## Objetivo

Criar uma página comercial para uma concessionária/loja de veículos, com foco em caminhonetes, atendimento, confiança, SEO local e contato com clientes.

## Tecnologias

- HTML
- CSS
- JavaScript
- Tailwind via CDN
- Lucide Icons
- GSAP
- Schema.org
- Vercel Speed Insights

## Destaques do projeto

- Estrutura de landing page comercial.
- Conteúdo voltado para venda de caminhonetes e veículos premium.
- Meta tags de SEO local para União da Vitória - PR.
- Dados estruturados `AutoDealer` com Schema.org.
- Organização modular em arquivos como `vehicles.js`, `reviews.js`, `validation.js`, `modals.js` e `site_config.js`.
- Testes básicos com `test_suite.js`.
- Política de privacidade no arquivo `privacy_policy.md`.

## Como rodar localmente

```bash
npm install
npm run dev
```

Depois acesse:

```text
http://localhost:3000
```

## Testes

